# Persistence (ATT&CK TA0003)

Scripts for enumerating the ways attackers survive a reboot: Run keys, scheduled tasks, services, WMI event subscriptions, startup folders, and LSA packages.

## Scripts

- **Invoke-PersistenceAudit** — enumerate all common persistence mechanisms, optionally diff against a JSON baseline, and flag unsigned or non-standard-path binaries. *(in-progress)*

See `SecurityAnalyst/README.md` for the full matrix.
