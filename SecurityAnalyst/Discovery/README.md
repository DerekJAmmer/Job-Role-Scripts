# Discovery (ATT&CK TA0007)

Hunt scripts focused on environment discovery — finding out what attackers are looking for when they're learning their way around a network.

## Scripts

- **Invoke-HuntPack** — orchestrator that runs the tactic-specific hunt scripts (LogonHunt, RemoteExecHunt, PersistenceAudit) against one or more hosts and produces a unified findings report. *(planned)*

See `SecurityAnalyst/README.md` for the full matrix.
