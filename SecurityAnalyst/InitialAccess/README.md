# Initial Access (ATT&CK TA0001)

Scripts for hunting how attackers first get in — logon anomalies, authentication patterns, and inbound entry vectors.

## Scripts

- **Invoke-LogonHunt** — hunt 4624/4625 events for rapid-burst failures (brute force), multi-source logons for the same account, and explicit-credentials logons (type 9). *(in-progress)*

See `SecurityAnalyst/README.md` for the full matrix.
